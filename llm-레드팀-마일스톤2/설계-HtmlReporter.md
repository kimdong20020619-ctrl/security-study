# 설계: HtmlReporter

허브: [[허브]]

## 무엇을
마일스톤1의 `ConsoleReporter`(터미널 텍스트)를 대체할 공유 가능한 정적 HTML 리포트 생성기.

## 왜
포트폴리오 제출·면접 시연용으로 터미널 캡처보다 브라우저로 바로 열어볼 수 있는 결과물이 훨씬 설득력 있음 — 원본 스펙이 M2 최종 목표로 명시.

## 어떻게
- 외부 라이브러리(Jinja2 등) 없이 표준 라이브러리 f-string만으로 HTML 문자열 생성 — 새 의존성 추가는 사용자 승인이 필요한 사안이라 이번 자율 작업 범위에서는 보수적으로 피함
- 파일 하나(`report.html`)에 CSS까지 인라인 — 첨부파일 하나로 공유 가능
- 구성: 요약 카드(총 시도/성공/차단/오류/판정불가 — [[설계-JudgeStatus리팩터링]] 4종 상태 모두 표시), 카테고리별 성공률 표, 페이로드별 상세 결과(상태별 색상 배지)
- `config.yaml`의 `reporter.output_path`로 출력 경로 지정 가능, 기본값 `report.html`
