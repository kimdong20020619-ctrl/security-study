# 세션 메모: Fable 5 / Opus 4.8 사이버 세이프가드 차단

허브: [[허브]]

## 무슨 일이었나
야간 작업 중 사용자가 `/model`로 Fable 5·Opus 4.8을 여러 번 시도했으나 계정 단위 "실시간 사이버 세이프가드"가 이 프로젝트(LLM 탈옥/레드티밍 관련 대화)를 반복적으로 차단함.

## 원인과 대응
로컬 `settings.json`에는 모델을 고정하는 설정이 없었음(직접 확인함) — 즉 로컬 설정 문제가 아니라 Anthropic 서버 단의 필터. 로컬에서 끌 수 있는 방법이 없어, 남은 작업 전체를 Sonnet 5로 진행함(이 모델은 같은 트리거가 없음).

## 남은 조치
정식 해제는 Cyber Verification Program 신청뿐(사용자 본인이 진행해야 함): https://support.claude.com/en/articles/14604842-real-time-cyber-safeguards-on-claude
