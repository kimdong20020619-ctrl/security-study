# 🎯 보안 포트폴리오 통합 로드맵 (2026.07 ~ 2027.12)

> **작성일**: 2026-07-10
> **기간**: 지금 ~ 2027년 12월 (약 18개월)
> **목표**: **2027년 졸업 직전(하반기) 비(非)관제 보안 신입 취업.** 구체 직무(보안 솔루션 SE / 사내 정보보안 / AI-adjacent)는 **2027 상반기 채용시장 보고 확정**. **AI 보안을 차별화 축**으로 얹음(첫 직무로 직접 노리진 않음 — 전문대졸 신입엔 비현실적). 취업 필수(SOC1·정보보안기사·포트폴리오 3개)는 직무 공통이라 그대로 준비 — 마스터 플랜은 아래 "📅 분기별 상세".

| 구분 | 현황 |
|------|------|

| **자격증** | 리눅스마스터 2급 · 네트워크관리사 2급 · CPPG |

## 🔍 취준 냉정 분석 (실측: Theori 취업가이드·채용공고)

| 직무 | 필수 역량 | 우대 자격증 | 포트폴리오 |
|------|----------|-----------|-----------|
| **보안관제(SOC)** | 네트워크 기초, SIEM 실습, 로그·패킷 분석, **교대근무** | 정보보안기사, 네트워크관리사 | 필수 아님, 있으면 차별화 |
| **침해대응(DFIR)** | 로그·아티팩트 분석, 악성코드 기초, **MITRE ATT&CK** | 정보보안기사, GREM | 실습·분석 경험이 차별화 |
| **보안컨설팅** | 관리체계·법규, 문서화 | **CPPG**, 정보보안기사 | 관리체계 산출물 강력 |

**냉정한 결론**
1. 관제 신입 실질 관문 = **정보보안기사 + 네트워크/로그 실력 + 교대근무 수용**. 네트워크관리사 2급은 우대 해당.
2. 프로젝트는 침해대응·컨설팅 **차별화 + 자소서 스토리**용. 1개 깊게 → [[미니홈랩SOC-구축가이드]]
3. **CPPG가 남과 다른 무기** — 개인정보/컴플라이언스 서브 트랙.
4. **과다 계획이 최대 리스크** — 완주율 > 개수. 하나씩 확실히.

## 🗂️ 올해(2026) vs 내년(2027) 분리

### ✅ 올해 (2026, 지금~12월) — 무료·학습·프로젝트 집중

> 알바 병행(주 3~4h) 기준 → **SEC1 완주 + SOC1 절반(모듈 1~6) + 홈랩 리서치·착수**가 현실적 목표.

- [ ] **학습**: SEC1 완주(~9월) → **SOC1 모듈 1~6** → [[TryHackMe-학습-노트화계획]]
- [ ] **프로젝트**: **미니 홈랩 SOC 리서치·착수** (완성은 2027 상반기) → [[미니홈랩SOC-구축가이드]]
- [ ] **대외활동**: 핵더챌린지 명지전문대 참여 여부 확인(6~7월) · KISA 취약점 신고포상제(상시) 감 잡기
- [ ] **대회(선택)**: DFC 2026 학생부(7/1~9/30) 문제풀이 경험 (stretch)
- [ ] **자격증 준비**: **정보보안기사 필기 공부 시작**(내년 응시 대비) · (옵션) 리눅스마스터 1급

### 🔜 내년 (2027) — 취업 트랙 (관제 신입, 하반기 취업 목표)

- [ ] **자격증**: **정보보안기사 필기(상반기) → 실기** (못 따도 필기 합격이라도 만들기)
- [ ] **학교**: 전공심화 과정 (졸업 직전)
- [ ] **학습**: SOC1 완주 → **SAL1 응시** (SOC2는 취업 후로)
- [ ] **프로젝트**: 홈랩 SOC 완성 + phishing·ioc-extractor 실데이터 → **대표 포트폴리오 3개 확정**
- [ ] **AI 차별화**: OWASP LLM Top 10 정독 + prompt-injection-guard 심화 (여력 시 TryHackMe AI1)
- [ ] **취업**: 상반기 이력서·자소서·포트폴리오 완성 → **Q2 말~Q3 비관제 보안 신입 지원·면접**(SE/사내보안/AI-adjacent)
- [ ] ⏸️ **후순위(취업 후로 미룸)**: 화이트햇 스쿨/BoB(6개월 과정) · SOC2 · 공모전·대외활동 다수 — 취업 준비와 시간 충돌

## 📅 분기별 상세 (18개월)

> **🎯 취업 데드라인**: 2027년 졸업 직전(≈2027 하반기) **비관제 보안 신입 취업**(보안 SE / 사내보안 / AI-adjacent — 상반기 확정). **AI 보안을 차별화 축**으로. 전용 AI 보안 직무 직접 진입은 전문대졸 신입엔 비현실적 → 기반 직무로 입사 후 이동.
> **페이스**: 알바 병행 = 평균 주 3~4h. → "다 하기" 불가, 취업에 필요한 것만 앞당겨 완성.

| 시기 | 학습·자격 | 프로젝트 | 취업 |
|------|------|------|------|
| **26 Q3(7~9)** | SEC1 완주 · 정보보안기사 필기 공부 시작 | — | — |
| **26 Q4(10~12)** | SOC1 모듈 1~6 | 홈랩 SOC 리서치 | — |
| **27 Q1(1~3)** | SOC1 완주+노트화 · **정보보안기사 필기 응시** · 전공심화 시작 | **홈랩 SOC 착수** | 이력서 초안 |
| **27 Q2(4~6)** | **SAL1 응시** · 정보보안기사 실기 | 홈랩 SOC 완성 + 도구 2개 실데이터 | 자소서·포트폴리오 완성 → **지원 시작** |
| **27 Q3(7~9)** | (여력 시 복습) | 스크린샷·마무리 | **지원 집중·면접·취업** |
| **27 Q4(10~12)** | — | — | 입사 or 계속 지원, 졸업 |

### 취업 위해 버릴 것 (냉정)
- ❌ **SOC2** → 취업 후로. 신입엔 불필요.
- ❌ **화이트햇 스쿨/BoB(6개월)** → 취업 준비·졸업과 충돌. 취업이 목표면 미지원(취업 후 역량강화용).
- ❌ **프로젝트 8개 깊게** → 대표 3개(홈랩 SOC + phishing + ioc-extractor)만.

### 취업에 실제로 먹히는 것 (우선순위)
1. 자격증: 네트워크관리사2급·CPPG(보유) + **정보보안기사**(있으면 강력, 없으면 필기라도)
2. **포트폴리오 3개**(홈랩 SOC + phishing + ioc-extractor) — GitHub + 스크린샷
3. **AI 보안 차별화**: prompt-injection-guard 심화 + **OWASP LLM Top 10** (+여력 시 TryHackMe AI1) — "AI에 관심+역량 있는 보안 신입"으로 어필
4. SOC1(+SAL1) 이수 + 학습노트

> 비관제라 **교대근무 불필요**(관제 대비 장점). 대신 진입 문이 관제보다 좁으니 **포트폴리오·자격·AI 차별화가 더 중요**.
