## 냉정한 전제 (왜 이 프로젝트인가)

- 실측 결과 **관제(SOC) 신입은 포트폴리오 필수 아님** — 자격증·네트워크 기초·교대근무가 관문.
- 그러나 **침해대응(DFIR)·컨설팅 차별화 + 자소서 스토리**로는 프로젝트 1개가 강력.
- 원칙: **여러 개 얕게보다 1개를 깊게.** 재현 가능한 문서화가 핵심.
- 방향: **방어(탐지) 중심** — 블루팀 진로와 일치.

## ✅ 메인 프로젝트 — 미니 홈랩 SOC (로그 기반 침해 탐지)

### 주제
격리 홈랩에 오픈소스 SIEM(Wazuh)을 구축하고, 표준 시뮬레이션 도구(Atomic Red Team)로 공격 기법을 재현한 뒤 **탐지 룰·MITRE ATT&CK 매핑**까지 수행하는 탐지 엔지니어링 프로젝트.

### 왜 최적인가 (근거)
- 취업가이드: SOC 신입 = SIEM 실습 + 로그/패킷 분석 + 공격 패턴 이해. DFIR = 로그·아티팩트 + MITRE ATT&CK. → **한 프로젝트가 전부 커버**.
- 리눅스마스터·네트워크관리사가 홈랩 구축 기반.
- TryHackMe SOC1 학습과 직결 → 자소서 일관성.

### 성공 기준
- [ ] Wazuh 매니저 + 에이전트(Ubuntu/Windows) 구동, 로그 수집
- [ ] 공격 기법 **5개** 시뮬레이션 → 탐지 알림
- [ ] MITRE ATT&CK 매핑 표
- [ ] GitHub README + 아키텍처 + 시나리오별 스크린샷 + 트러블슈팅 + 회고

### 기술 스택
VirtualBox · Ubuntu 22.04 · Windows 10 · **Wazuh 4.x**(SIEM) · Sysmon · **Atomic Red Team** · Markdown/draw.io

### 착수 시점
**2026 Q4** (SEC1·SOC1 진행 후). 주말 활용해 올해 완성 목표.

### 👉 단계별 실행
전체 설치·설정·탐지 시나리오·GitHub 구조·README 템플릿은 **[[미니홈랩SOC-구축가이드]]** 에 재현 가능하게 정리됨. 그 문서만 따라가면 완성.

## 🔁 대안/서브 프로젝트 — 개인정보(PII) 탐지·마스킹 + 미니 관리체계 (CPPG 활용)

### 주제
가상 기업의 개인정보 처리 흐름을 설계하고, **Python으로 PII(주민번호·전화·이메일·카드번호) 정규표현식 탐지·마스킹** 도구를 만든 뒤 **개인정보 처리방침·접근통제 정책·점검 체크리스트**를 산출물로 문서화.

### 왜 좋은가
- **CPPG를 실물 산출물로 증명** → 개인정보/컴플라이언스 컨설팅 차별화.
- 난이도 낮고 2~3주 단기 완성 가능.

### 성공 기준
- [ ] PII 5종 탐지·마스킹, 오탐/미탐률 기록
- [ ] 관리체계 문서 3종(처리방침·접근통제 정책·점검 체크리스트)
- [ ] GitHub README + 샘플 입출력

### 기술 스택
Python(`re`, `pandas`), Markdown. 선택: Streamlit UI.

### 착수 시점
**2027 Q2** (홈랩 프로젝트 이후, 단기).

## 📋 추가 프로젝트 후보 (직무별, 전부 방어·분석 중심)

> 포트폴리오는 **3개면 충분**. 서로 직무가 겹치지 않게 조합할 것. 홈랩 SOC를 확실히 끝낸 뒤 하나씩.

| 프로젝트 | 연결 직무 | 난이도 | 핵심 기술 | 왜 좋은가 |
|---------|----------|:---:|----------|----------|
| **피싱 이메일 분석기** | 관제·DFIR | ★★☆ | Python, .eml 파싱, VirusTotal API, IOC 추출 | SOC1 피싱 모듈 직결, 실무 그대로 |
| **로그 분석 자동화 도구** | 관제(SOC) | ★★☆ | Python, 정규표현식, 인증/웹 로그 | 브루트포스·스캐닝 탐지 |
| **허니팟 운영·분석** | 관제·위협인텔 | ★★☆ | Cowrie/T-Pot, 로그 분석 | 실제 공격 트래픽 수집 이력 |
| **위협 인텔(IOC) 수집·대시보드** | 위협인텔·CERT | ★★☆ | Python, 공개피드(abuse.ch), DB | Threat Intel + MITRE 매핑 |
| **PCAP 네트워크 트래픽 분석** | 관제·네트워크 | ★★☆ | Wireshark/Zeek | 네트워크관리사 강점 시너지 |
| **윈도우 이벤트로그·아티팩트 분석** | 침해대응·포렌식 | ★★★ | 이벤트로그, Autopsy | DFIR 지향이면 최적 |
| **CIS 벤치마크 보안설정 점검 스크립트** | 보안엔지니어·컴플라이언스 | ★★☆ | Bash/Python, 리눅스 하드닝 | 리눅스마스터 + 컴플라이언스 |

### 추천 포트폴리오 조합 (님 프로필 기준)
1. **미니 홈랩 SOC** (관제·SIEM 대표작) → [[미니홈랩SOC-구축가이드]]
2. **피싱 이메일 분석기** (관제·DFIR, 자동화, SOC1 직결)
3. **PII·관리체계(CPPG)** 또는 **허니팟** (차별화 축)

> 인프라(SIEM) + 자동화(스크립트) + 컴플라이언스(CPPG) 세 색깔이 나오면 서류에서 입체적으로 보인다.
