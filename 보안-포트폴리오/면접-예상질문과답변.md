# 🎤 면접 예상 질문 & 답변 포인트 (비관제 보안 신입)

> 답변은 **포인트**만 정리. 실제론 본인 말로, 프로젝트 사례를 근거로. 모르면 "아직 학습 중이지만 이렇게 접근하겠다"로 솔직하게.

## 1. 기술 기초
- **TCP/IP·OSI 설명?** → 계층 개념 + 방화벽이 어디서 동작하는지(네트워크관리사 기반).
- **대칭 vs 비대칭 암호화?** → 키 1개 vs 공개/개인키, HTTPS는 하이브리드(TryHackMe 학습).
- **리눅스에서 이상 프로세스 확인?** → `ps`, `top`, `netstat/ss`, 로그(`/var/log/auth.log`) 확인.
- **SQL 인젝션이 뭔가?** → 입력값이 쿼리로 실행되는 것 + 방어(파라미터 바인딩·입력검증).

## 2. 보안 개념
- **CIA 삼요소?** → 기밀성·무결성·가용성 + 각 침해 예시.
- **MITRE ATT&CK?** → 공격 전술·기법 프레임워크. 홈랩에서 탐지를 여기에 매핑함.
- **SIEM이 뭐고 왜 쓰나?** → 로그를 모아 상관분석·탐지·대응. 홈랩에서 Wazuh로 직접 구축.
- **개인정보보호법 기본?** → 최소수집·목적제한·안전조치(CPPG). pii-scanner 관리체계 문서로 실습.

## 3. 프로젝트 (가장 많이 물음)
- **가장 열정 쏟은 프로젝트?** → prompt-injection-guard STAR([[취업-이력서자소서-뼈대]] 5번).
- **오탐 45%를 어떻게 14%로?** → 강신호(DAN·개발자모드)와 일반 롤플레이('act as') 가중치 분리. `guard.py` 흐름 설명 가능해야.
- **왜 실데이터로 검증?** → 합성 예시론 통과하던 게 실제 데이터에서 결함 드러남. "돌려보길 잘했다".
- **홈랩 SOC 구축 경험?** → ⚠️ **실제 구축 후 답변**(안 했으면 "구축 계획·구성"까지만 솔직히).
- **프로젝트에서 협업?** → 개인 프로젝트지만 GitHub·이슈 관리·문서화로 협업 대비 습관.

## 4. AI 보안 (차별화 포인트)
- **AI 보안 왜 관심?** → 최신 AI 모델 탈옥 이슈 접하고 방어가 궁금 → 직접 가드레일 구현(솔직·자연스럽게).
- **프롬프트 인젝션 방어법?** → 입력 탐지(1차) + 출력 검증 + 권한 최소화 + 사람 승인(다층, OWASP LLM Top10).

## 5. 인성·직무
- **왜 보안인가?** → (본인 계기) + "직접 만들고 검증하는 게 재밌다".
- **비관제인데 관제 제안하면?** → (솔직히) 관심은 엔지니어링/사내보안이나, 필요하면 기초로 경험 가능.
- **강점?** → "이론→구현→실데이터 검증→개선"을 스스로 도는 것 + CPPG + AI 관심.
- **약점?** → 실무 경험 부족 → 홈랩·프로젝트로 메우는 중(구체 사례).
- **입사 후 목표?** → 기본기 빠르게 습득 → 중기 AI 보안 역량 추가([[취업-이력서자소서-뼈대]] 7번).
