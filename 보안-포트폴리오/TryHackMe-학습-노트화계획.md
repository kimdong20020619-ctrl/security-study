## 학습 축 (실측 확인)

**방향: 비관제(보안 엔지니어/SE·사내보안) + AI 차별화.** TryHackMe 갈래를 이 방향에 맞춰 선택:

| 갈래 | 결정 | 이유 |
|------|------|------|
| 보안 분석가(SOC) | **SOC Level 1까지만** | 방어 기초 + 홈랩 SOC 포트폴리오 기반 (SAL2·SOC2 스킵) |
| **보안 엔지니어(Security Engineer)** | ⭐ **메인 역할 경로** | 비관제 SE·사내보안·DevSecOps 직결 |
| 침투 테스터 | 스킵 | 공격 중심 → 방향 불일치 |
| AI 보안 | 차별화(여력 시 AI1) | "AI 관심+역량 보안 신입" 어필 |

**추천 순서**: SEC1 완주(진행 중) → **SOC Level 1**(방어 기초+홈랩) → **Security Engineer 경로**(Active Directory → DevSecOps → 하드닝 → **AWS 방어**) → **AI Security/AI1**(차별화).

> ☁️ **클라우드 포함 확정 (2026-07-14 방침 변경)**: 예전 "클라우드 제외" → **AWS 방어 1개 포함**(방어 관점, 얕게라도). 사내보안·SE에서 클라우드 수요가 커 "클라우드도 되는 신입"은 희소가치. (Azure 아닌 AWS로 — 국내 채용 수요 더 큼)
> 드림핵은 공격 중심이라 **선택 보조**(과투자 금지).

## 18개월 학습 일정

> **페이스 전제 (냉정·현실)**: 알바+학교+기타 병행 = 지속 가능한 보안 자기학습 **평균 주 3~4시간**(시험기간엔 0에 가까움). 방학 인텐시브는 단기만 유효.

| 시기 | TryHackMe (현실 목표) | 걸림돌 |
|------|------|------|
| **2026 Q3 (7~9월)** | SEC1 완주 (~9월 말, 17%→100%) | 알바·방학 |
| **2026 Q4 (10~12월)** | SOC1 모듈 1~5 | 2학기 13학점 + 기말고사 → 페이스 하락 |
| **2027 Q1 (1~3월)** | SOC1 완주 → write-up 노트화 | 방학 / 정보보안기사 필기 공부 시작 |
| **2027 Q2 (4~6월)** | **Security Engineer 경로 시작 (Active Directory)** | 전공심화 + 정보보안기사 실기 |
| **2027 Q3 (7~9월)** | Security Engineer(DevSecOps) · (여력 시) AWS 방어·AI Security | **취업 지원·면접** |
| **2027 Q4 (10~12월)** | 복습·부족분 보완 → 취업 마무리 | 자격증·지원 경쟁 |

### 냉정한 결론
- **2027 안 현실 목표 = SEC1 + SOC1 + 보안 엔지니어 경로 일부(AD·DevSecOps).** 전 과정 완주·SAL2·SOC2는 무리 → 취업 후로.
- 2027엔 **정보보안기사(최우선) + 홈랩 프로젝트 + 전공심화 + 취업준비**가 시간 경쟁. 주 3~4h로는 전부 불가 → 우선순위 필수.
- **비관제 보안 신입엔 SOC1 + 보안엔지니어 기초(AD 등) + 홈랩 SOC + AI 차별화**면 지원 경쟁력.
- 프로젝트도 8개 다 깊게 X → 홈랩 SOC + 1~2개만 실데이터로 깊게.
