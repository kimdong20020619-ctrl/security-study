# 모듈 2 — 컴퓨터 기초

## 컴퓨터 시스템 내부

- **메인보드 (Motherboard)** = 우리 몸의 골격 및 신경계 역할 / 다양한 부품을 자리에 고정하고, 서로 연결 / 여러 부품은 메인보드에 직접 꽂히거나 메인보드를 통해 연결된다.

- **CPU (Central Processing Unit)** = 프로세서라고 불리며, 우리 뇌의 일부에 비유할 수 있다. / 우리 뇌가 끊임없이 명령을 수행하는 것과 같은 역할 / 최근에는 여러 개의 코어를 통해 이를 병렬로 처리 / CPU 소켓을 통해 메인보드에 연결된다.

- **RAM (Random Access Memory)** = 우리 뇌의 단기 기억이나 작업 기억에 비유할 수 있다. / 우리는 관련 정보를 일시적으로 기억하는 역할과 동일 / CPU가 빠르게 접근해야 하는 데이터를 보관하는 역할 / 전원이 차단되면 저장된 모든 데이터가 사라지는 휘발성 메모리 / RAM 모듈은 속도와 성능을 높이기 위해 DDR5, DDR6 같은 기술을 사용

- **SSD/HDD (Storage)** = 우리의 장기 기억에 비유할 수 있는 저장 장치다. / 데이터를 저장 장치에 장기간 보관 / HDD는 저렴한 비용으로 대용량을 확보할 수 있어 널리 사용되지만 구형 기술을 기반으로 해 물리적으로 움직이는 부품이 있어 성능의 한계가 있지만, SSD는 움직이는 부품 없이 메모리 칩을 사용해 빠른 속도를 낼 수 있다. / SATA 케이블, PCI Express 슬롯을 통해 연결된다.

- **네트워크 어댑터** = 컴퓨터가 다른 시스템과 통신할 수 있게 해주는 역할을 한다. / 무선 방식과 유선 방식이 있다. / 대개 메인보드에 내장되어 있지만, 확장 카드 형태로 추가할 수도 있다. / 네트워크 카드는 일반적으로 PCI Express 포트를 통해 연결

- **PSU (Power Supply)** = 시스템의 모든 구성 요소에 에너지를 공급하는 역할을 한다. / 필수적인 부품으로 신중하게 선택해야 한다. / PSU의 공급 능력을 초과하는 전력을 필요로 할 경우, 시스템이 정상적으로 작동하지 않을 수 있다. / 콘센트에서 전력을 받아 메인보드용 주 커넥터나 Molex 커넥터 등 다양한 커넥터를 통해 각 부품으로 전력을 분배한다.

- **Graphics Card** = 운영 체제나 프로그램으로부터 정보를 받아 처리된 시각 데이터를 모니터로 출력하는 기능을 한다. / 메인보드의 PCI Express 슬롯에 장착된다.

- **Input/Output** = 입력 장치와 출력 장치이다. / 입력 장치로는 키보드/마이크/마우스/스캐너 등이 있다. / 출력 장치로는 모니터/프린터/스피커 등이 있다. / 주변 기기를 연결하는 일반적인 단자로는 USB/HDMI/DisplayPort 등이 있다.

## 컴퓨터 시스템 운영 체제 (부팅 과정)

### 1단계 — 전원 버튼 누르기
컴퓨터 시스템의 전원 버튼을 누르면 전원 공급 장치(PSU)로 신호가 전송되어 전원이 공급됩니다. 우리가 잠자는 동안 우리 몸의 전원이 꺼져 있다고 상상해 보세요. 잠에서 깨어나 산소를 공급받으면 우리 몸은 혈액을 펌핑하고 활동을 시작합니다.

### 2단계 — 펌웨어 시작
1단계에서 사용한 비유를 계속하자면, 신체가 시동되면 핵심 구성 요소들은 작동을 시작하지만 뇌는 아직 의식이 있는 상태가 아닙니다. 우리 몸과 마찬가지로 컴퓨터 시스템에도 모든 구성 요소가 시동될 수 있도록 하는 펌웨어가 있습니다. 이를 관리하는 중앙 시스템을 **통합 확장 가능 펌웨어 인터페이스 (UEFI)** 라고 합니다. (구버전은 BIOS라고 불렸으며, 현재는 UEFI로 대체됨)

### 3단계 — 전원 켜짐 자가 진단 (POST)
이제 우리 몸이 정상적으로 작동하기 시작했으니 모든 기능이 제대로 작동하는지 테스트할 차례입니다. UEFI는 **전원 켜짐 자체 테스트 (Power-On Self Test)** 를 실행하여 필요한 모든 구성 요소가 있는지, 올바르게 구성되었는지, 그리고 제대로 작동하는지 테스트합니다.

### 4단계 — 부팅 장치 선택
우리 몸이 제대로 구성되고 완전히 작동하기 시작하면, 시스템은 의식을 시작하기 위한 부팅 루틴의 위치를 찾습니다. UEFI는 운영 체제 부팅 루틴을 찾기 위해 어떤 장치를 먼저 찾아야 하는지 우선순위를 정한 목록을 유지합니다.

### 5단계 — 부트로더 초기화
이제 시스템은 의식이 위치한 뇌 부위를 알았으므로, 해당 부위를 시작하기 위한 로드 루틴을 실행합니다. 선택된 부팅 장치에서 **부트로더**가 초기화됩니다. 이 부트로더는 운영 체제(OS)를 선택된 부팅 장치에서 RAM으로 전송합니다. OS가 이체되면 UEFI는 다양한 구성 요소에 대한 제어 권한을 OS에 넘깁니다.
