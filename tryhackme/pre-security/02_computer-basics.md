# 모듈 2 — 컴퓨터 기초

---

## 컴퓨터 시스템 내부 구성요소

| 부품 | 역할 | 연결 방식 |
|------|------|-----------|
| **메인보드** | 골격·신경계 역할, 모든 부품 고정 및 연결 | — |
| **CPU** | 뇌의 일부, 명령 수행·병렬 처리 (멀티코어) | CPU 소켓 |
| **RAM** | 단기 기억, CPU가 빠르게 접근할 데이터 보관, 휘발성 (DDR5/DDR6) | RAM 슬롯 |
| **SSD/HDD** | 장기 기억, 데이터 장기 보관 / HDD: 대용량·저렴·느림 / SSD: 빠름 | SATA, PCIe |
| **네트워크 어댑터** | 다른 시스템과 통신 (유선/무선), 메인보드 내장 or 확장 카드 | PCIe |
| **PSU** | 모든 부품에 전력 공급, 초과 시 오작동 발생 가능 | Molex 등 |
| **그래픽 카드** | 시각 데이터 처리 후 모니터 출력 | PCIe 슬롯 |
| **Input/Output** | 입력: 키보드·마우스·마이크·스캐너 / 출력: 모니터·프린터·스피커 | USB·HDMI·DisplayPort |

## 컴퓨터 부팅 과정 (5단계)

### 1단계 — 전원 버튼 누르기
PSU로 신호 전송 → 전원 공급 시작

### 2단계 — 펌웨어 시작 (UEFI)
- 핵심 구성 요소 시동
- UEFI (구 BIOS): 모든 구성 요소가 시동될 수 있도록 관리하는 중앙 시스템

### 3단계 — 전원 켜짐 자가 진단 (POST)
- Power-On Self Test
- 모든 구성 요소 존재 여부, 설정, 정상 작동 여부 테스트

### 4단계 — 부팅 장치 선택
- UEFI가 우선순위 목록을 기반으로 OS 부팅 루틴이 있는 장치 탐색

### 5단계 — 부트로더 초기화
- 선택된 부팅 장치에서 부트로더 실행
- OS를 RAM으로 전송 → UEFI가 각 구성 요소 제어권을 OS에 넘김
